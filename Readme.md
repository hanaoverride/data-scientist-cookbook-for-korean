# 이 노트는 어떤 노트인가요?

1년동안 Data Science 공부를 하며 배운 것들을 정리할 노트입니다. 기초 컴퓨터 과학 지식부터 Data Science를 다루는데 필요한 다양한 주제의 글을 다룰 생각입니다.

**2025-07-13 EDIT**: 해당 노트 프로젝트가 거의 마무리되어가며 AI 기반 대량 리뷰와 함께 내용을 재편했습니다. 기존의 내용이 더 마음에 드는 분들이 있을것 같아 레가시 프로젝트의 링크 또한 함께 남깁니다.
